Babble Buddy

Embeddable chat widget for your apps

Demo

Click the chat bubble in the bottom-right corner. Try different themes:

Embed in Your App

<script src="YOUR_WIDGET_URL/babble-buddy.umd.cjs"></script>
<script>
  BabbleBuddy.init({
    appToken: 'your-app-token',
    apiUrl: 'your-agent-core-url',
    theme: {
      primaryColor: '#0f172a',
      backgroundColor: '#ffffff',
      textColor: '#1e293b'
    }
  });
</script>

Theme Options

{
  theme: {
    primaryColor: '#0f172a',    // Button & header color
    backgroundColor: '#ffffff', // Chat background
    textColor: '#1e293b',       // Text color
    fontFamily: 'system-ui',    // Font stack
    borderRadius: '12px'        // Corner rounding
  }
}
