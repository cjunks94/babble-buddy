Babble Buddy Widget

AI-powered chat widget you can embed anywhere

Click the chat bubble in the bottom-right corner to start a conversation!

Note: Make sure the Agent Core backend is running and you've set a valid app token.

Embed in Your App

<script src="YOUR_WIDGET_URL/babble-buddy.umd.cjs"></script>
<script>
  BabbleBuddy.init({
    appToken: 'your-app-token',
    apiUrl: 'your-agent-core-url',
    context: {
      app: 'my-app',
      instructions: 'Help users with...'
    }
  });
</script>

Configuration Options

{
  appToken: 'required - your app token',
  apiUrl: 'required - agent core URL',
  position: 'bottom-right | bottom-left | top-right | top-left',
  greeting: 'Initial message from assistant',
  context: { /* App-specific context sent to AI */ },
  theme: {
    primaryColor: '#6366f1',
    backgroundColor: '#ffffff',
    textColor: '#1f2937'
  }
}
